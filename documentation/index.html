Prema | Portfolio is Html5 + CSS3 portfolio template and works fine in all major browsers. It's powered by jQuery and you'll find nice and discreet interactivity. The code is clearly written and you will find comments for each section parts. Let's take a closer look at the structure of Html, CSS, JavaScript...

Folders and Files

When you unpack downloaded archive from themeforest.net you will find it as the following: 

  1. Documentation Folder:  Documentation for the convertible site template.
  2. Prema Folder:  Main folder of the template ( which contains folders and files of the template ).

Prema Folder Structure

- When you open Prema folder you will find folders as shown next:

     1 ) CSS folder   :  contains all CSS files used in the template.

     2 ) Fonts folder :  contains all fonts ( Icons fonts ) used in the template.

     3 ) img folder       :  where you should put your images in.

     4 ) Js folder       :  contains all JavaScript files used in the template.

      5 ) PHP folder : contains contact.php file used in the contact form.

     6 ) index-parallax.html   : Parallax index file.

     7 ) index-slider.html   : FullSize Slider index file.

     8 ) index-slider-parallax.html   : FullSize Parallax Slider index file.

     9 ) index-static.html   : Static index file.  ( Light Map )

     10 ) index-static-dark-map.html   : Static index file.  ( Dark Map )

     11 ) index-typed-text.html   : Typed Text index file.  ( static background )

     12 ) index-typed-text-fade-style.html   : Typed Text index file.  ( Fade Images Slider )

 

HTML Structure

Prema template is well commented to easily understand,

 in index.html you will find:

     <body>  area which contains the site and you will find that :

- Sections are sorted as their order on the site, so you can find any section you want easily.

PHP

In the PHP  folder, you will find the ( contact.php ) file which used in the contact form in the template.

All you have to do to make it works:

to change the email to your email open the php folder then contact.php file 

Find those lines here: 

// Change to your E-mail && Subject

$EmailTo = "admin@example.com"; // change to your E-mail here

then replace admin@example.com with your email and Prema-website with the subject you prefer.

Change Template Color

As I said before Prema Template has 16 different colors.

In the head section you will find this link: 

<!-- Color styles css file -->

<link rel="stylesheet" href="css/colors/color-1.css">

this means the default color is the color used in color-1.css file which is : #f7465b

if you want to change the default color of the template to another file just replace color-1.css in the previous link with color-*.css  ( use the number of css color file you prefer  from 1 : 16 ).

if you want to add your own color open css/colors/color-1.css and change f7465b with your own color ( change all f7465b in that file ). You can do that to any other file of the 16 colors files.

Then you will have to change the color in the color-box switcher in the index.html file :

<ul class="list-unstyled"> <!-- Colors boxes -->

     <li class="color-slot" data-background="#f7465b" data-target="css/colors/color-1.css"></li>

     <li class="color-slot" data-background="#874fe8" data-target="css/colors/color-2.css"></li>

     <li class="color-slot" data-background="#B39964" data-target="css/colors/color-3.css"></li>

     <li class="color-slot" data-background="#00b85b" data-target="css/colors/color-4.css"></li>

     <li class="color-slot" data-background="#0C7CD5" data-target="css/colors/color-5.css"></li>

     <li class="color-slot" data-background="#f9ca3b" data-target="css/colors/color-6.css"></li>

     <li class="color-slot" data-background="#f27768" data-target="css/colors/color-7.css"></li>

     <li class="color-slot" data-background="#38C8BD" data-target="css/colors/color-8.css"></li>

     <li class="color-slot" data-background="#ff5552" data-target="css/colors/color-9.css"></li>

you will have to change the data-background="#" to the color you choose. 

You can add or remove colors to the color-box switcher easily, for example, you want to add another color ( #faa )

1 ) Add another li like that: 

<li class="color-slot" data-background="#faa" data-target="css/colors/color-17.css"></li>

2 ) Change data-background to your color.

3 ) change the file name and url in data-target ( css/colors/color-17.css ).

4 ) Go to css/colors and create a new copy of color-1.css file and rename it to color-17.css and open it.

5 ) Replace all the f7465b with faa.

6 ) Done ;)

Parallax

Parallax Effect

To have that parallax effect I used parallax jquery plugin you can check it here : https://github.com/IanLunn/jQuery-Parallax

In scripts.js file you will find this code : 

/* ---------------------------------------------------

3 - Parallax Effect

----------------------------------------------------- */

if (/Android|webOS|iPhone|iPad|iPod|BlackBerry/i.test(navigator.userAgent)) {

    $("#home.parallax").css({"background-attachment": "scroll"});

    $("#fun-facts.parallax").css({"background-attachment": "scroll"});

    $("#quote.parallax").css({"background-attachment": "scroll"});

    $("#skills.parallax").css({"background-attachment": "scroll"});

    $("#testimonials.parallax").css({"background-attachment": "scroll"});

    $("#contact.parallax").css({"background-attachment": "scroll"});

    $(".slider-item.parallax").css({"background-attachment": "scroll"});

} else {

    $("#home.parallax").parallax("50%", 0.4);

    $("#fun-facts.parallax").parallax("50%", 0.4);

    $("#quote.parallax").parallax("50%", 0.4);

    $("#skills.parallax").parallax("50%", 0.2);

    $("#testimonials.parallax").parallax("50%", 0.2);

    $("#contact.parallax").parallax("50%", 0.1);

    $(".slider-item.parallax").parallax("50%", 0.5);

}

I made the hard work for you, as you can see I've assigned parallax effect to all sections with background images

all you have to do to give a section that parallax effect is to give that section a class="parallax" ( make sure that section is assigned to parallax effect in scripts.js as I showed up there )

For Example : 

this Testimonials section doesn't have parallax effect ( as it doesn't have parallax class ) :

<section id="testimonials" class="section text-center">

    ....

</section>

To make it parallax just give it parallax class like that : 

<section id="testimonials" class="parallax section text-center">

    ....

</section>

then check it out and enjoy! 

Images

Changing placeholder images to your images

As I mentioned before, Prema template doesn't come with images in the download files so, I used placeholder images from http://placehold.it

You can change these holders with your own images by using the same size as you find in the placeholder text link 

For exapmle : 

<img class="center-block img-responsive" src="http://placehold.it/800x600" alt="project-1">

As you can see here the best size to an image here to look perfect on website is 800x600 ( width = 800 xp && height = 600 px )

To change it to your image :

1- Put the image in the img folder.  ( let's say that image is pic-01.jpg )

2- Replace the src value http://placehold.it/800x600  with the image src which is : img/pic-01.jpg

So, it'll be like that: 

<img class="center-block img-responsive" src="img/pic-01.jpg" alt="project-1">

 

For Backgrounds: 

#about { background-image: url("http://placehold.it/1920x1280"; }

the best size of background images ( as in the example ) is 1920 x 1280

To change background to your image ( let's say it's name is bg-01.jpg )

replace the http://placehold.it/1920x1280  with : ../img/bg-01.jpg   it'll be like that : 

#about { background-image: url("../img/bg-01.jpg"); }

Dark or Light Map

Prema template has two different types of maps ( dark map or light map ) that you can use on your website .

In every index file you will find Google map section like that : 

<!-- Start Google Map Section -->

<section id="map" class="gmap"></section>

<!-- End Google Map Section -->

To use either light map or dark map you will find a script tag before body ending ( it has a class="map-script" ) just to know it 

<script class="map-script" src="js/light-map.js"></script>

as you can see it's the light map script tag. that will get you the light map in your website. To change it to dark map all you have to do is to include dark-map.js file inside js folder then replace src="js/light-map.js" with src="js/dark-map.js"

That's it, and you will have the dark map.
